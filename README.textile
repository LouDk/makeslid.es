h2. Kratek opis

Spletna aplikacija za izdelavo in predstavljanje html predstavitev.

h2. App page description

Predstavitev je odprta v browserju, nadzira pa se jo preko "control" pagea, ki je prilagojen za mobilne naprave oz web.
App bo imel FB in twitter login in share posameznih predstavitev.
Na control pageu vidiš kontrole (naprej, nazaj, gor, dol), timer, opombe za posamezne slide.
